The R codes in /simulx/ are the R codes used for the online user guide of Simulx:
http://simulx.webpopix.org/userguide/

The R codes in /mlxR/ are the R codes used for the online user guide of the following functions of mlxR:
  mlxplore.R
  pkmodel.R
  prctilemlx.R
  kmplotmlx.R
  catplotmlx.R
  monolix2simulx.R
  shinymlx.R
  exposure.R

Before running these R codes, install version 1.1.0 of the mlxLibrary from the Lixoft website: http://download.lixoft.com/?software=mlxlibrary

Before running the examples in /mlxplore/, install the Mlxplore software from the Lixoft website: http://www.lixoft.eu/products/mlxplore/get-mlxplore/

The following R libraries are required for running these R scripts:
    gridExtra
    reshape
    shiny
