The R codes in /simulx/ are the R codes used for the online user guide of Simulx:
http://simulx.webpopix.org/userguide/

The R codes in /mlxR/ are the R codes used for the online user guide of the following functions of mlxR:
  - catplotmlx
  - exposure
  - kmplotmlx
  - mlxplore
  - monolix2simulx
  - pkmodel
  - prctilemlx
  - readDatamlx
  - shinymlx
  - simpopmlx
  - statmlx
  - writeDatamlx

Before running these R codes, install the mlxLibrary from the Lixoft website: http://lixoft.com/downloads/

Before running the examples in /mlxplore/, install the Mlxplore software from the Lixoft website: http://lixoft.com/downloads/

The following R libraries are required for running these R scripts:
    gridExtra
    reshape2
    shiny
